# Genomas de referencia

Los archivos de genomas de referencia para _Candida auris_ no se incluyen en este repositorio debido a su tamaño. 

## Genomas necesarios

Este pipeline utiliza genomas de referencia para los distintos clados de _C. auris_:

- C.auris_CI_B8441.fasta y .gff3
- C.auris_CII_B11220.fasta y .gff3  
- C.auris_CIII_B11221.fasta y .gff3
- C.auris_CIV_B11243.fasta y .gff3
- C.auris_CV_B18474.fasta y .gff3

## Cómo obtener los genomas

Estos archivos se pueden descargar desde:

1. NCBI GenBank: https://www.ncbi.nlm.nih.gov/genome/browse/#!/eukaryotes/3685/
2. FungiDB: https://fungidb.org/fungidb/app/downloads/Current_Release/CaurisB11221/
3. Candida Genome Database: http://www.candidagenome.org/download/sequence/

Coloque los archivos descargados en esta carpeta (`resources/genomes/`) para que el pipeline pueda acceder a ellos.
