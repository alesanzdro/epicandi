# epicandi

## Project Overview

This project provides a robust bioinformatics pipeline for the identification of key antifungal resistance genes and their associated point mutations in Candida auris. The workflow is optimized to analyze whole-genome assemblies generated from Oxford Nanopore sequencing data.

The primary goal is to offer a rapid and accurate tool for genomic surveillance in clinical or research settings. It automates the detection of genetic markers linked to resistance against common antifungal agents, providing actionable insights into the resistance profile of a given C. auris isolate. This pipeline handles the post-assembly analysis, assuming a high-quality genome assembly has already been generated from the raw sequencing reads.
